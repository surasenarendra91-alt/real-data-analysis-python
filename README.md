## Project Description

This project performs sales data analysis using Python and Pandas.

The program loads a CSV dataset, cleans the data, performs statistical analysis, and generates a formatted sales report.

---

## Features

✔ Load CSV File

✔ Explore Dataset

✔ Check Missing Values

✔ Remove Duplicate Records

✔ Calculate:

- Total Revenue
- Average Sale
- Highest Sale
- Lowest Sale
- Best Selling Product
- Region Wise Sales

✔ Generate Final Report

---

## Technologies Used

- Python
- Pandas
- VS Code

## Installation

Install Pandas:

```bash
pip install pandas
```

Run the project:

```bash
python sales_analysis.py
```
