# Sales Data Analysis Report

## Project Overview

This project analyzes sales data using Python and the Pandas library.

---

## Objective

- Load the sales dataset
- Explore the dataset
- Clean the data
- Calculate important sales metrics
- Generate a sales report

---

## Dataset Information

Rows : 100

Columns : 7

Columns Included

- Date
- Product
- Quantity
- Price
- Customer_ID
- Region
- Total_Sales

---

## Data Cleaning

- Checked missing values
- Filled missing numerical values
- Removed duplicate rows

---

## Analysis Performed

- Total Revenue
- Average Sales
- Highest Sale
- Lowest Sale
- Total Quantity Sold
- Best Selling Product
- Region-wise Sales

---

## Technologies Used

- Python
- Pandas
- VS Code

---

## Conclusion

The project successfully analyzed the sales dataset using Pandas. It identified the total revenue, average sales, highest and lowest sales, best-selling product, and regional sales performance.
